# Number Base Converter (2-16)
![converter](https://github.com/user-attachments/assets/093b2371-9293-46ce-928e-9c8ae4f0b063)

Small class assignment to make a program that convert numbers between binary, octal, decimal and hexadecimal base. So I make this :) One of the challenges that I bind to myself on this project was to use just the <stdio.h> library and nothing more than my brain and fingers. Not sure if is the more optimized code make by mankid (quite sure isn't, to be honest) but works well, so that's it.

## Usage
Simply put the base of the number that you have and then the base that you want to convert to. After that, type your number and... end.

To compile, just use any common C compiler (I used GCC, but don't believe that by use another one you'll have problems).
